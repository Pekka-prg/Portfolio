AV-planning and -Maintenance for schools

I have done AV-planning and AV-maintenance for university. I was planning and taking in use Hybrid Active Class room with Sennheiser’s TrueVoiceLift feature. Signal was sent to displays with Crestron NVX AV over IP technology. System was steered by Crestron touch panel. Program for that was made by third party supplier as well as installations. My role was bug test and finetune the system with supplier. I have also done AV-layout design and draw AV-schematics using my own created AV-symbols. I'm familiar with Sennheiser’s TCC systems as well as many kinds of DSP systems that uses Dante audio. I'm Dante Level 3 Certified and also certified Crestron Technician. I'm familiar with Crestron tools and have used VTProe UI design tools as well as SIMPL and SIMPL+ programing tools.
